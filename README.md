## How to Add Your Photo

1. Save your professional photo in the Portfolio folder
2. Name it something like `profile-photo.jpg`
3. Open `index.html`
4. Find the section with the comment `<!-- Replace this div with your professional photo -->`
5. Uncomment the `<img>` tag and update the `src` attribute:
   ```html
   <img src="profile-photo.jpg" alt="Lokesh Kumar Saravanan">
   ```
6. Comment out or remove the `.photo-placeholder` div

## Customization

You can customize colors by editing the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
    /* ... more variables */
}
```
